# Circle Fund Landing Page

A premium, Vite + React landing page for the Circle Fund protocol.

## Prerequisites
- Node.js environment
- Vite

## Installation
1. Install dependencies: `npm install`
2. Define environment variables in `.env`:
   - `VITE_APP_ENTRY_URL` (URL of the App. Defaults to `/app`)
3. Run development server: `npm run dev`

## Included Sections
- Hero Section (Title, Subtitle, CTA to App)
- Problem Statement
- How It Works
- Key Features
- Benefits for Builders
- Transparency and Process
- FAQ
- Final CTA Section
